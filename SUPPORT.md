# Dukungan KosKeeper

KosKeeper adalah project open source yang dibuat untuk membantu pengelola kos/homestay. Jika kamu merasa terbantu, ada beberapa cara untuk mendukung pengembangannya:

## Donasi

Donasi seikhlasnya sangat membantu untuk:
- Pengembangan fitur baru
- Maintenance dan bug fix
- Dokumentasi yang lebih baik

## Kontribusi Non-Moneter

Donasi bukan satu-satunya cara untuk mendukung:

1. **Bug Report** - Laporkan bug yang kamu temukan
2. **Feature Request** - Usulkan fitur baru yang berguna
3. **Pull Request** - Kontribusi kode
4. **Dokumentasi** - Bantu perbaiki docs
5. **Share** - Sebarkan KosKeeper ke teman yang butuh
